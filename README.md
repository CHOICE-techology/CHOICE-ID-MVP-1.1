# CHOICE iD MVP 1.1
**Sovereign Identity Layer for CHOICE DAO**

CHOICE iD is a high-performance, decentralized identity (DID) engine. It serves as the foundational reputation and credentialing layer for the CHOICE ecosystem, bridging the gap between digital autonomy and immersive user experiences.

## 🚀 Key Features
* **Decentralized Identity (DID):** Sovereign profile management with multi-chain support.
* **Web3 Integration:** Powered by **Reown AppKit** (formerly WalletConnect), Wagmi, and Viem for seamless wallet connectivity.
* **High-Performance Architecture:** Refactored with **Vite** for optimized build times and efficient chunk handling.
* **Local-First Reputation:** Advanced reputation engine designed for future integration with PGLite and local AI (Ollama).
* **Phygital Aesthetic:** Implementation of the "Phygital White" design system (High-contrast typography + Neon accents).

## 🛠 Tech Stack
* **Framework:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **UI/UX:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
* **State & Data:** [TanStack Query](https://tanstack.com/query/latest) + [Lucide React](https://lucide.dev/)
* **Backend/Auth:** [Supabase](https://supabase.com/)
